# Docker Recap Quiz 

## Question 1
**Which command is used to create a Docker container but not start it?**
- A) `docker start`
- B) `docker build`
- C) `docker create`
- D) `docker run`  
✅ **Answer: C**

---

## Question 2  
**Which namespace isolates process IDs in Linux containers?**
- A) net
- B) ipc
- C) pid
- D) user  
✅ **Answer: C**

---

## Question 3  
**What does `docker image prune` do?**
- A) Removes all containers
- B) Deletes all Docker volumes
- C) Removes unused images
- D) Deletes running containers  
✅ **Answer: C**

---

## Question 4  
**What does the CMD instruction in a Dockerfile define?**
